This modified CP/M 3.0 BIOS features an Ethernet **TELNET I/O device** and a **RAMDisk**.

It is based on the work of agn453 (agn453/Z80-MBC2-master) on the Z80-MBC2 CP/M 3.0 BIOS.

The main changes are in files CHARIO.MAC (telnet device) and VDISK.MAC (Ram Disk).

Ready-to-use disk images are available in the Software/Disk_Images section.  

The ZPM3 version of the BIOS still provides the 61KB TPA achieved by agn453.
