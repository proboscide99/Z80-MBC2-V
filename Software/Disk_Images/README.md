Z80-MBC2-V is fully compatible with original project's disk images.  
You can prepare your sd-card using the original archive, then you may replace some files when you want to unlock specific features.  


| File | Description |
| :--- | :--- |
| **SD-S220718-R290823-v2** | Original Z80-MBC2 disk images |
| **DS2N00.DSK** | Replacement for CP/M 3.0 drive A: with custom BIOS (Telnet device, ramdisk) |
| **CPMLDR.COM** | Replacement for CP/M 3.0 loader (updates the release date printed at boot) |
| **DS2N07.DSK** | Replacement for CP/M 3.0 drive H: (custom BIOS development disk) |
